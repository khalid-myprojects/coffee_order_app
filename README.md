# ✨ Overview

**BREWED** is a modern and premium Flutter coffee ordering application designed with stunning UI/UX, smooth animations, glassmorphism effects, and responsive layouts.

This project showcases advanced Flutter frontend skills including:

- Beautiful modern design
- Hero animations
- Lottie animations
- Glassmorphism UI
- Animated navigation
- Responsive layouts
- Clean architecture
- Interactive cart system
- Smooth transitions

The app is built for portfolio, learning, and professional showcase purposes.

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/coffee_order_app.git
```

---

## 2️⃣ Open Project

```bash
cd coffee_order_app
```

---

## 3️⃣ Install Dependencies

```bash
flutter pub get
```

---

## 4️⃣ Run Application

```bash
flutter run
```

---

# 📦 Required Packages

```yaml
dependencies:
  flutter:
    sdk: flutter

  google_fonts:
  flutter_animate:
  lottie:
  glassmorphism:
  animated_bottom_navigation_bar:
```
